# Private Mode for AI

APL Sidecar is a minimal-exposure private session for sensitive AI work, not
another general-purpose LLM gateway.

The intended session is simple: start privately, preview and approve exposure,
split the minimum context between providers, reassemble locally, then choose
what to clean up and whether to retain a signed receipt.

> Don't give one AI the whole picture.

The OpenAI-compatible proxy is only a local integration surface. Receipts are
advanced trust evidence, not a prerequisite for understanding the first run.

"Private mode" is a product mental model, not a promise of zero traces. APL
cannot control provider retention and does not hide account identity, IP
address, or network metadata. Its defensible claims are minimal exposure,
local reassembly, and evidence of what the APL-controlled workflow disclosed.
